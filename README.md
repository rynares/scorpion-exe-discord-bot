# 👑 KING SCORPION BİLGİ PANELİ 👑

Bu proje, gücün ve kontrolün tek bir isim altında toplandığı gelişmiş bir yıkım sistemidir.
**Hata yapma.** Bu araç zayıflar için değil, krallar içindir.

---

## ⚙️ Kurulum Ritüeli

### 1. Sisteme Entegrasyon
```bash
npm install
```

### 2. Yapılandırma (`config.json`)
Kendi kimliğini ve hedef mesajını belirle:
```json
{
  "token": "BOT_TOKENINI_BURAYA_YAZ",
  "ownerId": "KENDI_DISCORD_ID_N",
  "prefix": ".",
  "nukeMessage": "Bu krallık artık Scorpion'a ait. Her şey sona erdi. @everyone"
}
```

### 3. Sistemi Uyandır
```bash
node index.js
```

---

## 📋 Yıkım Cephaneliği

| Komut | Etki Alanı |
|---|---|
| `.patlat` | Sistemin sonunu getirir, her şeyi silip Scorpion'un hakimiyetini kurar |
| `.ban` | Sunucudaki tüm varlıkları hiçliğe sürgüne gönderir |
| `.kick` | Zayıf halkaları sunucudan temizler |
| `.isimler` | Herkesin kimliğini siler ve 'Sistemin Kölesi' yapar |
| `.silici` | Sunucudaki tüm duygu ve ifadeleri (emoji/sticker) yokedir |
| `.kilit` | Tüm kanalları mühürler, sessizliğe gömer |
| `.nuke` | Bulunduğun kanalı yakıp yıkarak küllerinden yeniden yaratır |
| `.spam [mesaj]` | Kanala durmaksızın korku aşılar |
| `.dm <mesaj>` | Sunucudaki herkesin zihnine doğrudan mesaj iletir |
| `.rol` | Sunucuya 50 adet kan kırmızı yetkili rol enjekte eder |
| `.yetki` | Sana hak ettiğin mutlak gücü bahşeder |
| `.ayril` | İşin bittiğinde botu karanlığa geri yollar |
| `.panel` | Sistemi yöneteceğin ana komuta merkezini açar |
| `.yardim` | Tüm yıkım cephaneliğini listeler |

---

## ⚠️ Mutlak Kural

Bu botun hiç bir türlü satışı yapılamaz ve Botun Kullanımından kendiniz sorumlusunuz Rynares Development sorumlu değildir.
Bu sistem yalnızca sana aittir. Config içerisine yazdığın `ownerId` dışındaki hiç kimse bu komutlara erişemez. 
Sistemin çalışabilmesi için botun **Yönetici** yetkisine sahip olması ve rolünün en üstte olması gerekir.

> "Güç, ona sahip olmaya cesaret edenlerindir." - King Scorpion
